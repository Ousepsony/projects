    How to commit local changes to github.com

  1. open git bash and $cd into the required folder
  2. command git add to add and track the changes
  3. make sure you have acces by setting the global user id And email
  4. do $git commit -m "message" to commit the conges in the version control system
  5. (optional) if you want to see the changes in the remote repository, do $git push
  6. $git pull can be used to save any changes made on the remote repository to the local repository.
